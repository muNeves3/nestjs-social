## Description

A simple social media API made with NestJS framework, alongside with circleci and docker

#### Routes

- **Comment**: Create/Like/Delete/View
- **Post**: Create/Like/Delete/View
- **User**: Create/Edit/View/Delete/Follow

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
